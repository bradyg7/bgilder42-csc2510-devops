# CSC 2510: DevOps

This course introduces the principles and practices of DevOps, covering version control with Git, shell scripting, automation, continuous integration and delivery, containerization, and cloud infrastructure. Students gain hands-on experience with industry-standard tools and workflows used in modern software development and operations environments.
